# SAMPLE-ganttchart-kanban

React を使って、ガントチャートとカンバンを kintone に表示するカスタマイズです。

## Nodejs のバージョン

Nodejs >=14

## インストール

```console
npm i
or
yarn
```

## 開発＆テスト

```console
npm start
or
yarn start
```

初めて上記コマンドを実行する場合、Windows OS は証明書のインポートを求められます。Mac OS は sudo のパスワードの入力を求められます。
コマンドを実行した後、案内に従って、kintone の情報を入力してください。

## ビルド

```console
npm run build
or
yarn build
```
